在百度BAE下建立自己的hexo博客

hexo是个轻量级的静态博客,几乎可以在任何支持web服务下的服务器上运行,一般我们会把hexo放在
github 的github pages上面,充当一个免费无限量的服务器来运行。

但是介于在天朝,很多时候访问github都困难,不翻墙的情况下,浏览博客的速度会慢不少,所以我们考虑把hexo放到百度的BAE引擎上来运行。

  • 首先我们需要在BAE上新建一个部署,在这里我们选择Node-js.
